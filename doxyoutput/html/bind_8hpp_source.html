53  /*
54  * Binded function argumments are transformed to their values:
55  *
56  * - If the parameter is not a placeholder, the corresponding function call argumment is the
57  * parameter itself (Main specialization of 'transformation').
58  * - If the parameter is a placeholder, the corresponding function call argumment is the bind evaluation (call) parameter
59  * represented with that placeholder, that is, the argumment with index N-1, where the placeholder is _N
60  * (Partial specialization of 'transformation').
61  * Because tml::lists::get has O(n) complexity, the complexity of evaluating a tml::bind call is O(n^2) in the worst case.
62  * In practice is not, because tml::transform for lists is O(1), so the complexity is O(n) (See the documentation at "algorithm.hpp").
63  * Of course remember that template instantation depth is what that complexity measures.
64  */
65  template<typename T>
66  struct transformation : public tml::function<T>
67  {};
