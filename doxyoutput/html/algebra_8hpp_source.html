29 /*
30  * This header declares the set of basic algebra metafunctions. Like other functions, one sould evaluate
31  * them to retrieve the result.
32  *
33  * Also this header defines a set operators overloading which implement that algebra operations.
34  * That operators implement expression templates which compute the desired algebra operation.
35  * The result could be extracted via a decltype() aplication, or the macro ALGEVAL(), which is just
36  * a shorthand for decltype.
37  *
38  * Finally, this header defines some algebra primitives, customizable for different types.
39  */
40
